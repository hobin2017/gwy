# summary
- b(base基期)
- c(current现期)
- gr(growth rate增长率) = (c-b)/b
- 同比&环比

# 资料分析
### 基期&现期(用来对比基期的), 2题
- 同比增长10% => 比全年增长10%
### 增长量/增长率, 5题
- 增长量 = 现期量 - 基期量 = b*gr = c*gr/(1+gr)
- 增长率 = 增长量/基期量 
- 求增长率的问法: 增速, 增幅，高出多少倍
### 发展速度
- 发展速度 = 现期量/基期量 = 增长率 + 1
### 同比&环比
- 同比，比如2017年5月的同比，是以2016年5月作为基期
- 环比，比如2017年第三季度的环比，是以2017年第二季度作为基期；又比如2017年5月的环比，是以2017年4月作为基期
- 如果以一年作为一个周期，同比=环比

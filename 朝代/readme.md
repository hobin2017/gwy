- 960年，后周官员赵匡胤发动陈桥兵变。改国为宋。

宋朝生活
- 取消宵禁，夜市兴起，厨娘职业，地位尊贵，费用高昂。[百家讲坛20240403]
- 试种和推广印度的占城稻，粮食的增产，百姓生活从一日两餐到一日三餐。
- 炼铁提高，铁锅普及。厨具从陶罐到铁锅。饮食在煮肉和烤肉上，增加了炒菜（少量肉加菜）。西北草原和东北草原丢失后，养羊的地方少了，饮食变得清淡化和素食化。
- 制糖技术成熟。水果与糖制作成果脯。菜里加糖，由咸转甜。
- 开国初期，粮食不足，禁止酿酒。越南占城稻引进，粮食增产，允许酿酒，蒸馏限制，度数在5到20度，明朝酒可达50度。酒税1700万贯，占财政税收的40%。官营酒曲（原料），不许私营，小店从大店批发，税收抓大店，大店再收小店。[百家讲坛20240406]
- 宋瓷之首-汝瓷。颜色是按照宋徽宗的话 - 雨过天晴云破处。现科学结论是少量铁带来的天情色。[百家讲坛20240408]

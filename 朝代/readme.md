- 开国初期，粮食不足，禁止酿酒。越南占城稻引进，粮食增产，允许酿酒，蒸馏限制，度数在5到20度，明朝酒可达50度。酒税1700万贯，占财政税收的40%。官营酒曲（原料），不许私营，小店从大店批发，税收抓大店，大店再收小店。[百家讲坛20240406]

明朝2帝-朱允炆[百家讲坛20221212]
